# Event Scheduler

A modern web-based event scheduling system designed with a robust backend and a responsive frontend. This project combines cutting-edge technologies to provide seamless event management functionalities.

---

## **About the Project**

The **Event Scheduler** is a complete system for managing events, featuring:
- User registration and authentication.
- Creation, editing, and deletion of events.
- Validation to prevent overlapping events.
- Multi-user support, with events linked to their creators.
- RSVP functionality, allowing users to invite others to events and manage responses.
- Fully responsive design with a sleek "Matrix"-inspired theme.

---

## **Technologies Used**

### **Frontend**
- **React** with TypeScript: Provides a scalable and dynamic user interface.
- **React-Bootstrap**: Enhances UI design with responsive components.
- **Toastify**: Delivers elegant notifications for user interactions.
- **React-Router**: Handles route management efficiently.

### **Backend**
- **NestJS** with TypeScript: Offers a modular and scalable backend architecture.
- **PostgreSQL**: Ensures data integrity and relational database management.
- **TypeORM**: Simplifies database interactions with object-relational mapping.
- **JWT (JSON Web Tokens)**: Secures authentication and protects routes.
- **Class-Validator**: Provides robust validation for incoming data.

---

## **Features**

### **User Management**
- Secure user registration with field validations.
- JWT-based login and session management.
- Access control for protected routes.

### **Event Management**
- Add, edit, and delete events with dynamic forms.
- Prevent overlapping events with backend and frontend validation.
- Support for multi-day events.
- Invite other users to events with RSVP functionality.

### **RSVP**
- Users can accept or decline event invitations.
- List of events includes information about invitations and their statuses.

### **Design & User Experience**
- Matrix-inspired theme with green neon highlights.
- Fully responsive for optimal performance on different screen sizes.
- Calendar integration with clear visual indicators for today, events, and invites.
